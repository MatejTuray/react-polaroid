`Polaroid` (component)
======================

React component - Flippable Polaroid photo

Props
-----

### `backText`

Text on the back of the polaroid picture / card

type: `string`
defaultValue: `"Polaroid kitty - back"`


### `cardColor`

Optional background color prop

type: `string`
defaultValue: `"white"`


### `disabled`

Disable flipping

type: `bool`
defaultValue: `false`


### `flip`

Control flipping between front and back

type: `bool`
defaultValue: `false`


### `frontText`

Text on the front of the polaroid picture / card

type: `string`
defaultValue: `"Polaroid kitty - front"`


### `height`

Height

type: `number`
defaultValue: `400`


### `imgSrc`

Image uri

type: `string`
defaultValue: `placecat`


### `rotation`

Rotation of the component

type: `number`
defaultValue: `0`


### `type`

Flip animation type

type: `enum("side"|"default")`
defaultValue: `"default"`


### `width`

Width

type: `number`
defaultValue: `220`
